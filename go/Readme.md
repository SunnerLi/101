## Pokemen GO Kaggle dataset
### Predict the type of pokemen by given CP and HP

Usage
---

Train the model:
```
python main.py
```

Test by yourself:
```
$ python ask.py
Enter CP and HP in order: 1079 83
result:  Grass
Enter CP and HP in order: d

```

Show the data distribution:
```
python distribution.py
```
